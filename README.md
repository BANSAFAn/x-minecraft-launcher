## 🚀 Quick Start

### Download & Install

**Official Site**:  
👉 [Download XMCL for Windows, macOS, or Linux](https://xmcl.app)

**Windows (via Winget)**:
```bash
winget install CI010.XMinecraftLauncher
```

**macOS (via Homebrew)**:
```bash
brew tap voxelum/xmcl
brew install --cask --no-quarantine voxelum/xmcl
```

## ✨ Features

- 📥 **Smart Downloads**: Auto-install Minecraft, Forge, Fabric, OptiFine, Quilt, JVM from official or third-party mirrors.
- ⚡️ **Blazing Fast**: Multi-part downloads + HTTP/HTTPS socket reuse for maximum speed.
- 💻 **Cross-Platform**: Works on Windows 10/11, macOS, and Linux.
- 📚 **Multi-Instance**: Isolate game versions, mods, and settings per instance.
- 🗂 **Disk Efficient**: Uses hard/symbolic links to avoid duplicate mod files.
- 🔥 **Integrated Mod Platforms**: Built-in support for CurseForge & Modrinth.
- 📦 **Modpack Import/Export**: Fully compliant with CurseForge & Modrinth formats.
- 🔒 **Multiple Accounts**: Microsoft, Mojang, Ely.by, Littleskin.cn, and custom servers.
- 🔗 **LAN Over Internet**: Play multiplayer even if you’re not on the same physical network.
- 🔑 **Modern Packaging**: Code-signed Windows installers (`appx`, `appinstaller`) — no SmartScreen warnings!
